\# 🦟 Malaria Outbreak Prediction Dashboard



\## WHO AFRO Malaria Surveillance System



An interactive, AI-powered dashboard for monitoring and predicting malaria outbreaks in African countries using real WHO data.



\---



\## 🎯 Problem Statement



Malaria kills \*\*over 600,000 people annually\*\* in Sub-Saharan Africa, mostly children under 5. Health ministries lack real-time, actionable insights from existing WHO data to predict outbreaks and allocate resources effectively.



\## 💡 Solution



This dashboard provides:

\- \*\*Real-time surveillance\*\* of malaria cases across 10+ African countries

\- \*\*ML-powered outbreak prediction\*\* using weather data (temperature, rainfall, humidity)

\- \*\*Actionable recommendations\*\* for health ministries

\- \*\*Exportable insights\*\* for further analysis



\---



\## 📊 Features



| Feature | Description |

|---------|-------------|

| \*\*Country Filter\*\* | Analyze specific African countries |

| \*\*Year Filter\*\* | See trends over time (2018-2024) |

| \*\*KPI Cards\*\* | Total cases, deaths, positivity rate |

| \*\*Trend Chart\*\* | Monthly malaria case patterns |

| \*\*Bar Chart\*\* | Country comparison |

| \*\*Heatmap\*\* | Cases by country × year |

| \*\*Risk Gauge\*\* | Visual outbreak risk meter |

| \*\*Weather Sliders\*\* | Adjust temperature, rainfall, humidity |

| \*\*ML Prediction\*\* | AI-powered outbreak risk scoring |

| \*\*Data Export\*\* | Download filtered data as CSV |
