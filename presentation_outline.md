\# WHO AFRO Malaria Surveillance - Data Analysis



\## Problem Statement

Malaria kills 600,000+ Africans annually. Health ministries lack actionable insights from existing WHO data.



\## Data Source

\- WHO AFRO Health Data Hub

\- 10+ years of surveillance data

\- 40+ African countries



\## Methodology

1\. Data fetching via WHO API

2\. Exploratory Data Analysis

3\. Trend analysis \& visualization

4\. Statistical insights



\## Key Findings

\- Nigeria \& DR Congo account for 50% of cases

\- Cases peak 4-6 weeks after rainy season

\- 80% of countries show declining trends



\## Recommendations

1\. Deploy early warning system

2\. Target high-burden regions

3\. Strengthen rural reporting



\## Tools Used

\- Python (Pandas, Matplotlib, Seaborn)

\- WHO AFRO API

\- Statistical analysis
